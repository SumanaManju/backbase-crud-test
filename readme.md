# Sumana Anbukumar
Java application to test the CRUD operations of the computer website

 

## Installation

 

Use the package manager maven to install the application

 

```bash
mvn clean install
```

 

## Test Scenarios
1. Can CREATE a computer with all details including Computer Name, Introduced Date, Discontinued Date and Company using ‘CREATE’ form

2. Can READ the details of the computer Computer Name,Introduced Date, Discontinued Date and Company using 'READ' form

3. Can UPDATE the Computer Name,Introduced Date, Discontinued Date and Company using ‘UPDATE’ form
	
4. Can DELETE the computer using 'DELETE' form
